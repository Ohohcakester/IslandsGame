# Islands Game
A little puzzle game. It is NP-COMPLETE!

The game can be played here: https://ohohcakester.github.io/IslandsGame/

# Instructions

* Collect all the **coins** (Yellow) and reach the exit to clear the stage!
* Collect **energy** (Cyan) to warp between islands. You need 1 energy for each warp.

### Movement Controls

* **Arrow Keys**: Movement
* **Space**: Warp

### Camera Controls

* **Z**: Zoom (Hold)
* **Shift**: Zoom (Toggle)

### Game Controls

* **R** or **ESC**: Restart Game
